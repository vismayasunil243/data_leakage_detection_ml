# Data Leakage Detection using Machine Learning

This project builds a machine learning model to detect abnormal user activities that may indicate potential data leakage.

## Technologies Used
- Python
- Scikit-learn
- Pandas
- NumPy
- Flask

## Machine Learning Models
- Logistic Regression
- Random Forest
- Decision Tree
- svm

## Project Steps
1. Data preprocessing
2. Exploratory Data Analysis
3. Feature engineering
4. Model training
5. Model evaluation

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score
